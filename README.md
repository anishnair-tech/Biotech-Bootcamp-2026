# Biotech-Bootcamp-2026
A personal repository for learning Python-based bioinformatics, exploring molecular visualization in PyMOL, and automating lab protocol workflows.

Right now, DNA_Sequence_Analysis.ipynb contains a full Central Dogma Workflow that processes a raw DNA sequence to calculate its length and GC content, transcribes it into RNA, translates it into an amino acid sequence, and computes the final protein's molecular weight.
